# m06a_ETA - 台灣國道 AI 雙模型 ETA 預測系統

這是一個結合機器學習與國道電子收費門架拓撲資料的專業行程時間預測系統。

## 🚀 技術架構
本系統採用**雙模型聯動邏輯**，以達成精準的行程時間預算：
1.  **流量預測 (Random Forest)**：首先根據時間（小時、星期）、門架位置與歷史流量，使用隨機森林模型預測目標路段的即時交通量。
2.  **速度指標轉換**：將預測流量轉換為速度指數 (Speed Index) 與平均車速。
3.  **精確 ETA 預算 (LightGBM)**：將上述指標結合路段長度、車輛類型等特徵，交由 LightGBM 模型計算出最終的預估耗時 (ETA)。

**系統組成：**
-   **後端**：使用 **FastAPI** 構建高性能非同步 API，並實作**拓撲尋路演算法 (Topology Routing)** 進行鏈結接龍。
-   **前端**：結合 **Leaflet** 互動式地圖，視覺化呈現路徑預測結果與地理資訊。

## 📂 目錄結構
-   `main.py`: FastAPI 主程式與 API 進入點，負責路由分發與靜態檔案掛載。
-   `backend/predictor.py`: 核心 AI 預測邏輯、雙模型推論與拓撲尋路演算法。
-   `data/`: 存放國道門架座標、里程牌價表及 `gantry_topology_final.parquet` 拓撲資料。
-   `models/`: 存放 `.joblib` 與 `.pkl` 模型權重（已由 Git LFS 託管）。
-   `static/index.html`: 互動式地圖使用者介面，提供起訖點搜尋與預測結果顯示。

請確保您的 Python 環境已安裝必要套件：

